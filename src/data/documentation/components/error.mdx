## Introduction

The `Error` component is a user interface (UI) element that displays to the user errors that occured during some process, i.e submitting a form etc. 
## Anatomy

The `Error` component consists of the following elements:

- `Error` HTML text element

## Demonstration

## Usage

To use the `Error` component, simply import it from `@/components/ui/error` and use it as follows:

<CodeBlock
  title="Basic import"
  fit={true}
  snippet={`
    // components
    import Error from '@/components/ui/error'

    const Page = () => {
      
      return (
        <div className='flex items-center justify-center'>
            <Error
              title='Email Error'
              message:'The email couldn't send!'
            />
        </div>
      )
    }

    export default Page
    `}
/>

## Props

Below is a table of props for use within the `Error` component.

| Prop                                             | Type                              | Default                                      | Description                                                                                       |
| ------------------------------------------------ | --------------------------------  | -------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| <TableProp type='solid'>title</TableProp>        | <TableProp>String</TableProp>     | <NoWrap>"Error"</NoWrap>                     | The title of the error component. Default is set but you can supply any title you like.           |
| <TableProp type='solid'>message</TableProp>      | <TableProp>String</TableProp>     | <NoWrap>"Something went wrong!"</NoWrap>     | The message of the error component. Default is set but you can supply any message you like.       |
| <TableProp type='solid'>icon</TableProp>         | <TableProp>Bool</TableProp>       | <NoWrap>true</NoWrap>                        | Default state for whether or not an icon is displayed.                                            |
| <TableProp type='solid'>size</TableProp>         | <TableProp>String</TableProp>     | <NoWrap>"default"</NoWrap>                   | The size of the checkbox. Possible values are `sm` , `default` and `lg` .                         |
| <TableProp type='solid'>className</TableProp>    | <TableProp>String</TableProp>     | <NoWrap>" "</NoWrap>                         | CSS class name(s) to apply to the error element.                                               |


## Full Code

Below is the full code for `@/components/ui/error`.

<CodeBlock 
title='Completed component'
snippet={`
// base
import { AiOutlineExclamationCircle } from "react-icons/ai";

// utils
import mergeClasses from "@/utils/helpers/mergeClasses";
import { cva } from "class-variance-authority";
import clsx from "clsx";

const Error = ({
  icon = true,
  title = "Error",
  message = "Something went wrong!",
  size = "default",
  className
}) => {
  // styling
  let baseErrorStyles = cva(["flex", "items-start", "gap-1", "text-red-500", "transition-all"], {
    variants: {
      size: {
        sm: "text-sm",
        default: "text-base",
        lg: "text-lg",
      },
    },
    defaultVariants: {
      size: "default",
    },
  });

  return (
    <div className={mergeClasses(baseErrorStyles({ size }), className)}>
      {icon && (
        <AiOutlineExclamationCircle
          className={clsx("flex-shrink-0", size === "sm" ? "mt-1" : "mt-[5px]")}
        />
      )}
      <p>
        <span className="font-medium">{title && 'title: '}</span>
        {message}
      </p>
    </div>
  );
};

export default Error;
`}/>
