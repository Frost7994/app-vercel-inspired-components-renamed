## Introduction

The `Button` component is a UI element used to trigger an action, such as submitting a form or navigating to another page. It can be styled to fit the design of your application, and supports various props to control its behavior and appearance.

## Anatomy

The `Button` component consists of the following elements:

- `Button` HTML element
- Inner text or child component

## Usage

To use the `Button` component, simply import it from `@/components/ui/button` and use it as follows:

<CodeBlock
  title="Basic import"
  fit={true}
  snippet={`
    // components
    import Button from '@/components/ui/button'

    const Page = () => {
      return (
        <div className='flex flex-col justify-center items-center gap-2'>
            <h1 className='font-bold text-4xl'>The Example Page</h1>
            <Button onClick={() => {
                alert('Hello using the button')
            }}>Click Me</Button>
        </div>
      )
    }

    export default Page
    `}

/>

## Functionality

The button is built using **Class-Variance-Authority** and **TailwindCSS**. It is a simple UI component used to trigger events.

## Props

Below is a table of props for use within the `Button` component.

| Prop                                          | Type                            | Default                         | Description                                                                                                                                                                                            |
| --------------------------------------------- | ------------------------------- | ------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| <TableProp type='solid'>children</TableProp>  | -                               | <NoWrap>null</NoWrap>           | Child elements that will be rendered inside the button element.                                                                                                                                        |
| <TableProp type='solid'>type</TableProp>      | <TableProp>String</TableProp>   | <NoWrap>" "</NoWrap>            | Defines the type of button. Possible values are `button` , `submit` or `reset` . By default, no type attribute is included in the generated HTML, so the button behaves like a regular button element. |
| <TableProp type='solid'>onClick</TableProp>   | <TableProp>Function</TableProp> | <NoWrap>{`( ) => { }`}</NoWrap> | The click event handler function.                                                                                                                                                                      |
| <TableProp type='solid'>size</TableProp>      | <TableProp>String</TableProp>   | <NoWrap>"default"</NoWrap>      | The size of the button. Possible values are `sm` , `default` and `lg` .                                                                                                                                |
| <TableProp type='solid'>style</TableProp>     | <TableProp>String</TableProp>   | <NoWrap>"default"</NoWrap>      | The visual style of the button. Possible values are `default` , `outline` and `ghost` .                                                                                                                |
| <TableProp type='solid'>variant</TableProp>   | <TableProp>String</TableProp>   | <NoWrap>"default"</NoWrap>      | The visual colors of the button. Possible values are `default` , `primary` and `secondary` .                                                                                                           |
| <TableProp type='solid'>disabled</TableProp>  | <TableProp>Bool</TableProp>     | <NoWrap>false</NoWrap>          | Disables the button.                                                                                                                                                                                   |
| <TableProp type='solid'>loading</TableProp>   | <TableProp>Bool</TableProp>     | <NoWrap>false</NoWrap>          | Shows a loading spinner in beside the button label.                                                                                                                                                    |
| <TableProp type='solid'>fullWidth</TableProp> | <TableProp>Bool</TableProp>     | <NoWrap>false</NoWrap>          | Sets the button to full width.                                                                                                                                                                         |
| <TableProp type='solid'>className</TableProp> | <TableProp>String</TableProp>   | <NoWrap>" "</NoWrap>            | CSS class name(s) to apply to the button element.                                                                                                                                                      |
